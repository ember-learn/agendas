# agendas
Agendas for learning core team face-to-face meetings

Modeled after the all-hands core teams face-to-face meetings [agenda](https://github.com/ember-project/agendas).

- Agendas are organized by year, and then by month. 
- agenda items should be submitted as a PR to the specific agenda. 
- team members should be prepared to present supporting materials for their agenda item at the face-to-face meeting.
